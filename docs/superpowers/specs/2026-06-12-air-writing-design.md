# 隔空写字 — 实时手势识别交互系统 设计文档

**日期：** 2026-06-12
**状态：** 设计完成，待实现

---

## 1. 概述

通过 HTML + JavaScript + MediaPipe 构建单文件应用，在浏览器中运行。用户通过摄像头实时检测手势，实现隔空写字、擦除、缩放、消散等功能。

**核心体验：** 摄像头画面为背景，霓虹发光笔迹叠加显示，环形 UI 面板通过手势唤出。

---

## 2. 技术选型

| 项目 | 选择 |
|------|------|
| 手势检测 | MediaPipe HandLandmarker (`@mediapipe/tasks-vision@0.10.18`) |
| 加载方式 | ESM import from jsDelivr CDN |
| 渲染 | 单 Canvas 叠加 Video |
| 架构 | 单 rAF 主循环（检测→状态→渲染） |
| 文件 | 单文件 HTML，0 依赖 |

---

## 3. 手势映射（最终确认版）

| 手势 | 判定条件 | 触发动作 |
|------|---------|---------|
| 🖐️ 整个手掌 | 5 指全部伸展，持续 0.5s | 打开/关闭 UI 面板 |
| ✌️☝️ 食指+中指合拢 | 食指+中指伸展，两指尖距离 < 0.05，其余卷曲 | 写字（笔迹跟随两指尖中点） |
| ☝️ 食指指向 | 仅食指伸展，其余卷曲，悬停 0.8s | 选择 UI 菜单项 |
| 🤙 拇指+食指张开 | 拇指+食指伸展，两指尖距离 > 0.2，其余卷曲 | 触发雪花消散 |
| ☝️☝️ 双手食指 | 两只手各伸出食指 | 双指缩放（0.3x~3x） |

**防抖：** 手势需连续 6 帧确认；UI 悬停 0.8s 带进度圆环；手掌 UI 切换有 1.5s 冷却。

---

## 4. 系统架构

```
浏览器窗口（全屏 + 响应式）
├── <video>  摄像头画面（CSS scaleX(-1) 镜像）
├── <canvas> 绘制层（CSS scaleX(-1) 镜像）
│   ├── 笔迹线条（霓虹发光效果）
│   ├── 消散粒子（雪花飘落动画）
│   ├── 手势光标指示器
│   └── 环形 UI 选择面板
└── 底部状态栏（当前工具 | 颜色 | 大小）

主循环 (requestAnimationFrame):
  1. handLandmarker.detectForVideo() → 手势数据
  2. 手势识别引擎 → 状态机更新
  3. Canvas 清屏 + 绘制笔迹 + 粒子 + UI
```

---

## 5. 环形 UI 面板

### 布局（水平展开式）

```
[🧹橡皮] [✏️笔] [🎨色块×6] [━━ 大小滑块 ━━]
```

### 交互
1. 手掌张开 → 面板从顶部滑入（玻璃态半透明背景）
2. 食指悬停目标按钮 → 进度圆环填满（0.8s）
3. 圆环填满 → 选中（闪烁/放大反馈）
4. 手掌再次张开 → 面板滑出隐藏

### 工具项
- ✏️ 笔：默认工具，写字模式
- 🧹 橡皮：擦除模式，光标为 30px 半透明圆
- 🎨 颜色：霓虹蓝 `#00d4ff`、品红 `#ff2d55`、绿 `#32d74b`、金 `#ffd60a`、紫 `#af52de`、白 `#ffffff`
- 📏 大小：3 档（细 3px / 中 6px / 粗 10px）

---

## 6. 写字 & 橡皮擦

### 写字
- 笔迹点 = 食指 TIP(8) 和中指 TIP(12) 的中点
- 二次贝塞尔曲线平滑连接
- 霓虹渲染：外层粗线半透明发光 + 内层细线纯色
- 每笔存储为独立 Stroke 对象（世界坐标）

### 橡皮擦
- 30px 半透明圆形光标跟随指尖中点
- 遍历 Stroke，标记被橡皮圆擦过的为删除
- 删除时 0.3s 渐隐过渡

---

## 7. 双手缩放 & 雪花消散

### 双手缩放
- 两食指间距变化 → 缩放比例（0.3x ~ 3x）
- 缩放中心 = 两指尖中点
- 任一手消失则退出缩放，保持当前值

### 雪花消散
- 触发：拇指+食指张开
- 所有 Stroke 采样点 → 雪花粒子
- 粒子：向下飘落 + 水平正弦摆动 + 衰减透明
- 全部消失后自动结束

## 8. 数据模型

```javascript
// 笔迹
Stroke = {
  id: string,
  points: [{x, y, timestamp}],
  color: string,
  size: number,
  tool: 'pen' | 'eraser'
}

// 粒子
Particle = {
  x, y, vx, vy,
  life: 0~1, decay: 0.005~0.015,
  size: 2~6, color: string,
  wobbleOffset: random
}

// 视图状态
ViewState = {
  scale: 1,         // 0.3 ~ 3
  offsetX: 0, offsetY: 0
}

// 应用状态
AppState = {
  tool: 'pen' | 'eraser',
  color: string,
  size: number,
  uiOpen: boolean,
  gesture: 'idle' | 'writing' | 'pointing' | 'scaling' | 'dispersing',
  strokes: Stroke[],
  particles: Particle[]
}
```
